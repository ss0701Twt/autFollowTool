フォローの設定


–

Twitter規約による凍結回避のため、入力した最小値から最大値までの間でランダムの秒数でフォローしていきます。


入力した人数で、フォローをやめます。



画像設定していないアカウントはフォローしません。






入力した値以上のフォロワーがいる人のフォローをスキップ


入力した値以上のフォロー中がいる人のフォローをスキップ


フォロー上限 に達した時に入力した秒数フォローを待機


ISO 639-1 言語コードをコンマ区切りで入力すると、他の言語のツイートはスキップされます。たとえば、日本語のツイートのユーザーのみをフォローしたい場合は、jaを使用してください。すべての言語のツイートからユーザーをフォローするには、このフィールドを空のままにします。注：このオプションは、ツイートからフォローする場合（検索ビューなど）にのみ適用されます。


コンマ区切りでユーザ名を入力すると、フォローをスキップします。

–

Twitter規約による凍結回避のため、入力した最小値から最大値までの間でランダムの秒数でフォローしていきます。


入力した日数以内にフォローした人のフォロー解除をスキップします。この項目は、Ace Massfollowでフォローした人に対してしか適用されません。


入力した人数で、アンフォローをやめます。





コンマ区切りでユーザ名を入力すると、フォローをスキップします。

Autopilot(自動操作)

Autopilotは一連のアクションを実行し、一定時間停止後にそれらを繰り返します。 2つ以上のアクションを定義するとすぐに、Twitterホームページでオートパイロットを開始できます（Twitterページの更新が必要になる場合があります）。上記で定義されたMassFollow、MassUnfollowのオプションが適用されます。

設定方法デモムービー（Superpower本家のやつ）はこちら

No. Type URL リミット* Idle timeout (seconds)**

* You can also overwrite the limit for each action.

** If no successful follow/unfollow/like/retweet could be made for x seconds, continue with the next action. Note: Mass Follow/Unfollow/Like/Retweet Interval (seconds) does not count as idle. Leave this field empty to set no idle timeout.


Repeat autopilot after x minutes. Leave this field empty if you do not want to repeat the autopilot.
